blobs (2019)

When the program starts, it initiates a camera output layered underneath a number of particle blobs that represent the same shapes Victor Pasmore uses in his paintings. From the live camera input, a refreshing image is used in a particle class to create a number of particles that replace the actual pixels in the output. The blob particles are created by first collecting colours from his paintings, this is done in the program itself by using an external library, ColourScheme. For each colour a particle is created each with randomly assigned width and height within a range. The painting particles are then outputted on top of the live camera output so as to create the idea of an immersive experience within a Pasmore painting. A push button connected to the raspberry pi is used to capture a frame and these are saved accordingly. Another input of interactivity is the mouse; when moved on its x-axis the width of the webcam particles change and on the far-left they become almost lines. Sound is also outputted which uses this same variable to change its frequency. The idea was to create a poster that immerses the viewer looking at it and creates his own interpretation of a Pasmore painting.
